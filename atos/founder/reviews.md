## Daily Executive Review (15–30 min)

1. Read Daily Brief (Atlas).  
2. Mission check — any drift?  
3. Clear or defer Decisions Needed.  
4. Capture journal note if material (R901) — operational, not Canonical.  
5. Confirm tomorrow’s top priority.

## Weekly Planning / Review

1. Review REG-PROJ in-progress projects.  
2. Risks & incidents (Sentinel input).  
3. ATOS milestone gate status.  
4. Promotion queue glance (REG-PROMO-Q).  
5. Reprioritize if needed (Founder decision recorded).  
6. File weekly review note (template REF-R1108).

## Quarterly Review

1. Identity alignment (SPEC-002 / legacy identity corpus).  
2. Strategy vs outcomes.  
3. Executive performance / charter fit.  
4. ATOS version goals.  
5. Approve or reject major roadmap shifts (ADS if architectural).

## Decision recording

Material Founder decisions → decision register / ADR as applicable (STD-001). Atlas ensures traceability.
